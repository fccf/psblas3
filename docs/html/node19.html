CNV -- convert internal storage format

Type:
Asynchronous.
On Entry
desc
the communication descriptor.
Scope: local.
mold
the desred integer storage format.
Scope: local.
Specified as: a object of type derived from (integer) vbasedatapsb_T_base_vect_type.
